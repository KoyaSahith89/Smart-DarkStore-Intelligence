## 🎓 How to Use

### **Quick Start** (5 minutes)
```bash
# 1. Setup
pip install -r requirements.txt

# 2. Initialize & populate
python main.py  # Choose options 1, 2

# 3. Start API
python -m uvicorn api.main:app --reload

# 4. Start Dashboard
streamlit run dashboard/app.py
```

### **Demo Flow** (10 minutes)
1. Show Executive Dashboard (1-2 min)
2. Demonstrate problem solutions (3-5 min)
3. Run financial simulation (2-3 min)
4. Summary & Q&A (1-2 min)

### **Customization**
- Edit `config.py` for parameters
- Modify CSS in `dashboard/app.py` for branding
- Replace `simulate_data.py` with real data
- Adjust alert thresholds in config

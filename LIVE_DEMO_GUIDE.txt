🐛 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

❌ Can't see dashboard?
   ✓ Check: http://localhost:8501
   ✓ If not opening, restart: streamlit run dashboard/app.py

❌ Dashboard shows error?
   ✓ Refresh browser (Ctrl+R)
   ✓ Clear browser cache
   ✓ Check API is running: http://localhost:8000/docs

❌ API not responding?
   ✓ Check: http://localhost:8000/docs
   ✓ If down, restart: python -m uvicorn api.main:app --reload

❌ No data showing?
   ✓ Make sure data simulation ran
   ✓ Check: python data_simulator/simulate_data.py

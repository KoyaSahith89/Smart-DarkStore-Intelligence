## 📌 Overview

AI-powered analytics platform for dark store and quick-commerce operations.

Helps operations teams to:
- 📊 Monitor store performance  
- 📈 Forecast demand  
- 📦 Optimize inventory  
- 🚚 Predict delivery time  
- 📍 Simulate new store locations  
- 💰 Analyze profit impact

## 🛠 Tech Stack

Python • Streamlit • Pandas • NumPy • Scikit-learn • SQL • Plotly
